Simple Nybble Substitution (nsub) Program
Written by: Joshua Johnson

Reads from stdin, writes to stdout.

Flags:
    -e for encrypt (default)
    -d for decrypt
    -k <filename> for file containing the key
    -x if the key file is in ASCII-coded hexidecimal
    -b if the key file is in binary (default)


Environmental Expectations:
    This program has been tested and runs on the storm.cise.ufl.edu bastion ssh server.
    The CISE computers on the server are running Ubuntu 18.04.1 LTS (GNU/Linux 4.15.0-29-generic x86_64).

Bugs:
